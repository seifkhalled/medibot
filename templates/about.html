About This Project

The AI Medical Chatbot is an intelligent virtual assistant designed to provide accurate and reliable medical information in real-time. It uses a Retrieval-Augmented Generation (RAG) system that combines a vector-based document retrieval with NLP processing to answer user queries effectively.

Key Features:

  • Provides concise, evidence-based answers to medical questions.
  • Detects critical symptoms and issues urgent red-flag warnings.
  • Offers references and external resources for further reading.
  • Handles various medical topics, including diseases, treatments, and preventive care.
  • Maintains a user-friendly and interactive chat interface.

Technical Overview:

  • Backend: Python & Flask for handling requests and responses.
  • NLP & Retrieval: HuggingFace embeddings, custom NLP pipelines, and LangChain for retrieval-based generation.
  • Vector Database: Pinecone for storing and querying document embeddings.
  • Frontend: HTML, CSS, and JavaScript for a responsive chat interface.
  • Knowledge Sources: Gale Encyclopedia of Medicine, curated medical literature, and domain-specific articles.

References & Resources:

  • National Cancer Institute (NCI)
  • Centers for Disease Control and Prevention (CDC)
  • MedlinePlus
  • Mayo Clinic

This project was developed as part of the NTI NLP Summer Training to demonstrate practical applications of AI in healthcare, providing users with accurate information while emphasizing the importance of consulting licensed clinicians for personalized advice.
